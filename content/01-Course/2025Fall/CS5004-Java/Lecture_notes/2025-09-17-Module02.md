# Methods 
### Methods parts
Every **method** definition consists of the following parts:
1. A purpose statement. This consists of a single sentence that describes the purpose of this method. A longer description of the method may follow. (
	- `// Returns the larger of two numbers`
2. The type of the value returned from the method, known as the _return type_
	- `int`
	- `String`
	- `void` 
3. The method name, where the standard naming convention starts with a lowercase letter and uses **camelCase** to distinguish words within the name
	- Example: `getUserName`, `parseJson`, `toString`, `isEmpty`
4. A parenthesized *argument list*, consisting of the type and name of each argument, separated by commas
	- (int a, int b, String name)
5. The *method body*, surrounded by braces;`{}` this is the code to execute when the method is invoked
	- `{}`
### Object in `String` form
sample
```java
@Override 
public String toString(){
	return this.firstName + " " + this.lastName;
}
```

Structure of class:
```java
public class Book {

  // field
  private String name;
  private String author;
  private int price;

  public Book(String name, String author, int price) {
    this.name = name;
    this.author = author;
    this.price = price;
  }

  public String getName() {
    return this.name;
  }

}
```
### Extending a Class with Methods
```java
public float salePrice(float discount) { 
  return this.price - (this.price * discount) / 100; 
}
```

### Methods with objects as arguments
```java
public boolean same(Person other) { 
  return this.firstName.equals(other.firstName) && 
  this.lastName.equals(other.lastName) && 
  this.yearOfBirth == other.yearOfBirth; 
}
```
test
```java
Person p1 = new Person("Alan", "Wang", 1996);
Person p2 = new Person("Alan", "Wang", 1996);

boolean result = p1.same(p2);
// this = p1, other = p2
System.out.println(result); // true
```
- In Java, `String` is an object (reference type).
	- `==` compares whether **two references point to the same memory address**.
	- `.equals()` compares the **contents of the strings**.

### Methods Returning Objects
```java
  public Book discountBook(float discount) {
    float discountedPrice = this.salePrice(discount);
    return new Book(this.title, this.author, discountedPrice);
  }
  
  @Test
  void testDiscountBook() {
    pat = new Person("Pat", "Conroy", 1996);
    Book original = new Book("Beaches", pat, 20);
    Book discounted = original.discountBook(25); // 25% off
    // Assert
    assertEquals("Beaches", discounted.getTitle());  // Title unchanged
    assertEquals(pat, discounted.getAuthor());  // Author unchanged
    assertEquals(15, discounted.getPrice(), 0.01); // Price should be 15.0
  }
```
The discounted book keeps the original book’s title and author. Only its price is adjusted.
