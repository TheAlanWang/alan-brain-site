## Constructor
- Definition: In java, constructor is a special method used to create and *initialize an object*.
- When we use the `new` keyword, the constructor is automatically called.
```java
public class Student{
    private String name;
    private int age;
    
	public Student(){
		this.name = "unknwon";
		this.age = 0;
	}
}
```
- `public class Student` → anyone can use this class.
- `public Student()` → anyone can create a Student object using `new Student()`.
*encapsulation* private name age
