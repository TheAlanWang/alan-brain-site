## Typical Class Definition
```java
// 1. Class (基本结构)
public class Animal {
    // Fields (封装的属性)
    private String name;

    // Constructor
    public Animal(String name) {
        this.name = name;
    }

    // Method (封装的行为)
    public void makeSound() {
        System.out.println("Some sound...");
    }

    public String getName() {
        return name;
    }
}

// 2. Inheritance (继承)
public class Dog extends Animal {
    public Dog(String name) {
        super(name); // 调用父类构造器
    }

    // Polymorphism (多态：重写方法)
    @Override
    public void makeSound() {
        System.out.println("Woof!");
    }
}

// 3. Abstraction (抽象类 & 接口)
abstract class Shape {
    abstract void draw(); // 抽象方法
}

interface Movable {
    void move();
}

// 4. Using OOP
public class Main {
    public static void main(String[] args) {
        Animal a = new Dog("Buddy"); // 多态：Animal 引用指向 Dog
        a.makeSound(); // 输出 "Woof!"
    }
}

```
