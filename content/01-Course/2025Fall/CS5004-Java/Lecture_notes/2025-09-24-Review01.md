```java
public class hello{
	public static void main(String[] argus){
		System.out.println("hello");
	}
}
```

- Q: What is `(String[] argus)`?
	- an array of Strings named args that holds whatever values the user typed after the program name when starting the program.
	- `java HelloWorld Alan 123 CS5004`
	
- Q: From `.java` to running on JVM
	- step1: Compile with `jvm` -> `javac HelloWorld.java`
	- step2: Run with java -> `java HelloWorld`
## Variable
- Variable Naming
	- CamelCase eg:`myAge`
	- Must start with a letter, `_`, or `$` (not a number).
	- Cannot be a Java keyword (`int`, `class`, `static`, etc.).

- Different **Data Types**:
	- byte → short → int → long → float → double → char → boolean
	1. byte: 4-bit
	2. short: 16-bit
	3. int: 32-bit
	4. long: 64 bit
	5. float: 32-bit
	6. double: 64 bit
	7. char: 16-bit
	8. boolean: true or false

- Q: How to declare and initialize a float/long/char/boolean variable?
	- `long a = 100000l`
	- `float a = 0.0001f`
	- `char a = 'A'` -> `char` must be exactly one character inside single quotes `' '`.
		- `char a = '\u00A9';` -> ©
	- `boolean isWeekend = true` / `boolean isWeekend = false`

- Q: What is *`String`*?
	- `String` is a class
	- `String alan = "Alan"`
### Data type conversion
- Two types of data conversion
	- **Implicit (widening)** → automatic, safe
	- **Explicit (narrowing / casting)** → manual, may lose data
