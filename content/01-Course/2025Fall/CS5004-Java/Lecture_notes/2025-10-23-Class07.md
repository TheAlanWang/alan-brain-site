#### Employee.java (Interface)
```java
  int count(Predicate<Employee> condition);
```
- `Predicate<Employee>`: 
	- a function that takes an `Employee` as input, and returns a `boolean` value.
	- This is a **functional interface type**: `Predicate<Employee>` 

**Apply a yes/no test (boolean condition)** to objects of type `T`.
That includes:
- Filtering a list
- Counting items that match some rule
- Finding something that satisfies a condition
- Deleting only items that meet a certain criterion

```c
count(e -> true);
count(e -> e.getGender() == Gender.FEMALE);
count(e -> e.getAnnualPay() > 100000);
```


`Predicate<Employee>`: Automatically creates an anonymous class instance

#### supervisor.java
```java
  private List<Employee> supervisee;
  
  @Override
  public Employee addSupervisee(String supervisorName, Employee supervisee) {
    if (this.name.equals(supervisorName)) {
      this.supervisee.add(supervisee);
      return this;
    }
    for (int i = 0; i < this.supervisee.size(); i++) {
      this.supervisee.set(
              i,
              this.supervisee.get(i).addSupervisee(supervisorName,
                      supervisee));
    }
    return this;
  }
```

- `supervisee` is `List<Employee>`
- `this.supervisee` is `List<Employee>`
