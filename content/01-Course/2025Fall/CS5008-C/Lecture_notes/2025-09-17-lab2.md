## Pointer
```c
int value = 50;
int *pointer = &value;
```
- `pointer` **holds the address** of `value`.
- `*pointer` **dereferences** the pointer, giving the actual value stored in `value`.

## Pointer and array
```c
int arr[5] = {10, 20, 30, 40, 50};
int *p = arr;   // int *p = &arr[0];

array[0] = 96; // do the same thing
pointer[0] = 96;

int *left = arr;     // equal 
int *left = &arr[0];
```

## Memory
```
┌─────────────────────────┐
│   Static Memory (Data)  │ → Global & static variables
│                         │   Lifetime: entire program
├─────────────────────────┤
│   Heap                  │ → Dynamically allocated memory (malloc/free)
│                         │   Grows upward ↑
│                         │   Lifetime: until free is called
├─────────────────────────┤
│   Stack                 │ → Local variables, function calls, parameters
│                         │   Grows downward ↓
│                         │   Lifetime: until function returns
└─────────────────────────┘

```

| Part              | Stores                          | Lifetime               | Managed by          |
| ----------------- | ------------------------------- | ---------------------- | ------------------- |
| **Static Memory** | Global & static variables       | Entire program runtime | Compiler/runtime    |
| **Stack**         | Local variables, function calls | Until function returns | Automatic (system)  |
| **Heap**          | Dynamically allocated objects   | Until `free` is called | Programmer (manual) |

### Create a tic tac toe

```c
    char board[9] = {0};
```
- `char board[9]` → creates an array of 9 elements, each of type `char`.
- `= {0}` → initializes the **first element to 0**, and all the remaining elements default to 0 as well.

### Printing addresses inside the function is different
- In a function parameter, `char board[9]` is actually treated as `char *board`.
- What’s passed is just the address of the first element, not the whole array object.

### function: `fgets`
```c
char *fgets(char *str, int n, FILE *stream);

// sample
char buffer[8] = {0};
fgets(buffer, sizeof(buffer), stdin);
```
A **buffer** is a temporary storage area in memory.
