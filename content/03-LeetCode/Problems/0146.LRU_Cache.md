# Question
### Restate the problem
- put: (key, val), no return
- get: give key, return val if no key, return -1

When capacity is exceeded, evict the least recently used (LRU) item.

Q: no duplicate key

## Method - Linked list
### Approach
key insight: 
- `dic = {key:node}`
- `class node` with key, val, prev, next
- use linked list to record key

head -> key1 -> key2 -> tail

Initial:
- `dic = {key: node}`
- linked list: head to tail

`_add:`
	cur = head.next
	node.prev = head
	node.next = cur
	cur.pre = node
	head.next = node

`_remove:`
	node.next = nxt
	node.prev = pre
	pre.next = nxt
	nxt.prev = pre

`get:`
	check if in the dic
	after get, _remove(node), _add(node)

put:
- if exist:
	- `_remove(key)`
	
`dic[key] = val`
`_add(key)`
- if over capacity:
	- `_remove(last_node)`
	- remove dic
### Complexity
- Time Complexity: O(1)
- Space Complexity: O(1)
### Edge Case

### Code
```python
class Node:
    def __init__(self, key, val):
        self.key = key # for eviction
        self.val = val
        self.prev = None
        self.next = None

class LRUCache:

    def __init__(self, capacity: int):
        self.cap = capacity
        self.dic = {}

        self.head = Node(-1, -1)
        self.tail = Node(-1, -1)
        self.head.next = self.tail
        self.tail.prev = self.head
    
    def _add(self, node):
        cur = self.head.next
        
        node.prev = self.head
        node.next = cur

        cur.prev = node
        self.head.next = node
    
    def _remove(self, node):
        # head -> key1 -> key2 -> tail
        pre = node.prev
        nxt = node.next
    
        pre.next = nxt
        nxt.prev = pre

    def get(self, key: int) -> int:
        if key in self.dic:
            self._remove(self.dic[key])
            self._add(self.dic[key])
            return self.dic[key].val
        else:
            return -1

    def put(self, key: int, value: int) -> None:
        if key in self.dic:
            self._remove(self.dic[key])

        node = Node(key, value)
        self.dic[key] = node
        self._add(node)
        
        if len(self.dic) > self.cap:
            remove_node = self.tail.prev
            self._remove(remove_node)
            del self.dic[remove_node.key]
```
