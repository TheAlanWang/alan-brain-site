# Question
### Restate the problem
- Given a String
- Return `int`, the length of the longest substring without repeating characters.
---

## Method 1: Sliding window
### Approach
1) **Expand** the window by moving `right` and adding `s[right]`
2) While the window is invalid (a duplicate exists), 
	- **Shrink** by moving `left` and removing `s[left]`
3) **Update** the max length after the window becomes valid
### Complexity
- Time Complexity: O(n)
- Space Complexity: O(n)
### Edge Case
- if not `s`, return 0
### Code
```python
from collections import defaultdict

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        dic = defaultdict(int)
        n = len(s)
        left = right = 0
        res = 0
        while right < n:
            dic[s[right]] += 1
            while dic[s[right]] > 1:
                dic[s[left]] -= 1
                left += 1
            res = max(res, right - left + 1)
            right += 1

        return res
```
