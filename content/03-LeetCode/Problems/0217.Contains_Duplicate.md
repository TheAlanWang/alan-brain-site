# Question
### Restate the problem
- Given an integer array nums, return boolean. Return True if any value appears at least twice, otherwise return False.
# Method
### Approach: 
Approach: Use a set to track seen numbers.
- If we ever see the same number again, it's a duplicate.
### Complexity
- Time Complexity: O(n)
- Space Complexity: O(n)
### Edge Case

### Code        
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        seen = set()
        for num in nums:
            if num not in seen:
                seen.add(num)
            else:
                return True
        
        return False
```
